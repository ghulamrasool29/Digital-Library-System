# 📚 Digital Library Management System

## 📖 Project Overview

The Digital Library Management System is a desktop-based application developed using Python and Tkinter. It provides an easy and interactive way to manage library records through a Graphical User Interface (GUI).

The system allows users to add books, search books by title or author, borrow books, return books, and delete books efficiently.

---

## ✨ Features

- Add New Books
- Search Books by Title
- Search Books by Author
- Borrow Books
- Return Books
- Delete Books
- View Library Records
- User-Friendly GUI

---

## 🛠 Technologies Used

- Python 3
- Tkinter
- Object-Oriented Programming (OOP)
- VS Code

4. Open Terminal.
5. Run:

